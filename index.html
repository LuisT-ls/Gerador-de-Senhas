      book Glossário de Segurança

      Senha Forte:
      Uma senha longa e complexa, contendo uma combinação de letras maiúsculas e minúsculas, números e símbolos. É difícil de adivinhar e resistente a ataques de força bruta.
      Ataque de Força Bruta:
      Um método de ataque que tenta descobrir uma senha testando sistematicamente todas as combinações possíveis de caracteres até encontrar a correta.
      Phishing:
      Um tipo de ataque cibernético em que os invasores tentam enganar as vítimas para que revelem informações confidenciais, como senhas, números de cartão de crédito ou dados pessoais, geralmente por meio de e-mails, mensagens ou sites falsos.
      Keylogger:
      Um software malicioso que registra as teclas digitadas pelo usuário, incluindo senhas, e envia essas informações para os invasores.
      Autenticação de Dois Fatores (2FA):
      Uma camada extra de segurança que exige que o usuário forneça duas formas de identificação para acessar uma conta, como uma senha e um código enviado por SMS ou gerado por um aplicativo.
      Gerenciador de Senhas:
      Um software que armazena e gerencia senhas de forma segura, permitindo que o usuário utilize senhas fortes e diferentes para cada conta sem precisar memorizá-las.
      Hashing de Senha:
      Um processo que transforma uma senha em um código criptografado (hash) que não pode ser revertido para a senha original. Isso protege as senhas armazenadas, pois mesmo que um invasor acesse o banco de dados, ele não terá acesso às senhas em texto claro.
      Ataque de Dicionário:
      Um tipo de ataque de força bruta que utiliza uma lista de palavras comuns, nomes e outras informações para tentar adivinhar a senha de um usuário.
      Engenharia Social:
      Uma técnica de manipulação psicológica utilizada por invasores para enganar as vítimas e obter informações confidenciais, como senhas ou dados pessoais.
      Malware:
      Um software malicioso projetado para causar danos a um sistema de computador, roubar dados ou realizar outras atividades maliciosas. Exemplos de malware incluem vírus, worms, trojans e ransomware.
